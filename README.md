piles = `[3,6,7,11]`, 
H = 8

4

Приклад 2:
piles = `[30,11,23,4,20]`,
H = 5

Приклад 3:
piles = `[30,11,23,4,20]`,
H = 6

23

Важливо:
1 <= piles.length <= 10^4
piles.length <= H <= 10^9
1 <= `piles[i]` <= 10^9
